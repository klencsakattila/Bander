# Bander

#How to use/ setup

Download the full zip

or

1. Clone the Repository
  git clone https://github.com/yourusername/bander.git
  cd bander

2. Install Dependencies
  2.1 Frontend
      cd frontend
      npm install
   
  2.2 Backend
    cd backend
    npm install

3.Running both 
  3.1 Start the backend server:
    cd backend
    npm run dev

  Backend should now run on: http://localhost:3000

  3.2 Start the frontend
    cd frontend
    npm run dev
